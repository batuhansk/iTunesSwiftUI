**iTunesSwiftUI**

A sample app that demonstrates the fundamentals of SwiftUI and Combine.

- Fundamentals of **Reactive**
- Building a simple **Networking Layer** by using **Combine**
- **SwiftUI** & **UIKit** Representation Proxy (**UIViewRepresentable**)
